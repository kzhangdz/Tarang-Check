# Tarang Check
[![Flutter][flutter-badge]][flutter-url] [![Dart][dart-badge]][dart-url]

A Flutter-based cross-platform mobile application to provide a centralized source for viewing geocoded events of singers from Thailand.

## Explore
View events on a map interface, with the following features:
- Events on the current day highlighted with a star.
- Past events that are grayed out.
- A table view to view all events on the map.
- A filter to select a date range and desired artists.

## Artists
Select from a wide variety of artists and view all their current events. 

Artist pages feature custom-tailored colors to bring out each group's unique personality.

## Events
View event details, with options to open the original source in Instagram, open the location in Google Maps, or add the event to the system calendar. The user can display the original schedule text or translate into their native language.
